# Agromart Client

## Run

1. Copy `.env.example` to `.env`.
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

## Frontend Features

- Responsive landing page with featured and hot deals sections
- Product search, category filtering, pagination
- Cart and checkout with location-based input
- User dashboard with profile and order tracking countdown
- Admin dashboard for analytics and order status updates
- In-app socket notifications with sound alerts
